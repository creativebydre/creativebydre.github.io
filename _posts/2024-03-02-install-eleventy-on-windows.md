### Download Node Js

For run eleventy 11ty project , you need to [download node js ‚Üí](https://nodejs.org/en/download)

Download and run installation node js in to your devices.

### Eleventy Starter Project

Now you can download [eleventy 11ty](https://11ty.dev) starter project, [download eleventy 11ty starter now ‚Üí](https://www.hockeycomputindo.com/themes/eleventy/)

### Run Eleventy Project

Next step you can extract eleventy starter project in to your project folder - open terminal and run `npm run dev` and open `localhost:8080`
